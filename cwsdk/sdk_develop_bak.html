触网插屏sdk嵌入

一、嵌入sdk

    在您的项目中导入cwcp_xxx.jar，
  • 在eclipse工程中，把cwcp_xxx.jar拷贝到libs目录下，
  • 在Android Studio 2.x版本中，将工程目录结构改为Project模式，然后将jar包拷贝到libs文件夹里面，如果没有libs文件夹则新建libs文件夹之后再拷贝进去，拷贝进去之后选则Build→Rebuild Project选项即可

三、sdk初始化

  • 在应用启动的时候加入如下代码来初始化广告SDK
    • aar.getInstance(this).init( String key,String channelid);（尽量添加在程序最先执行的地方） 参数context为应用当前的上下文。
    key：从广告后台获取
    channelid：发布app渠道

四、API接口

  • 应用内显示插屏一次
    • aar.getInstance(this).rqw(getApplicationContext());

五、注意事项

  • 目前该sdk支持Android2.3或以上版本
  • 手机要有网络,能通电话
  • 手机要挂载有Sdcard
  • 请使用a作为测试key
